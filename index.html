Sobre mí

Me llamo Jean Mosquera, soy un Data Analyst apasionado con una misión clara: usar los datos para resolver problemas complejos y generar estrategias de negocio efectivas.

Especializado en el uso de Python y SQL para la gestión y análisis de datos, trabajo con herramientas como Tableau y Power BI para crear dashboards interactivos.

Me adapto rápidamente a nuevas tecnologías y metodologías. Mi objetivo es contribuir a empresas innovadoras donde la analítica de datos sea clave para tomar decisiones informadas.
